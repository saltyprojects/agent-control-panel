# Agent Control Panel

**Visual dashboard for managing AI agent teams**

## The Problem

Teams running multiple AI agents face:
- No visibility into what agents are doing
- No cost tracking across agents
- No permission controls
- No easy way to spawn/kill agents
- No emergency stop button

## The Solution

Agent Control Panel gives you:
- **Real-time monitoring** - See all running agents at a glance
- **Cost tracking** - Total spend across all agents
- **Permission management** - Control what each agent can access
- **One-click controls** - Spawn new agents or kill runaway ones
- **Live activity logs** - Know exactly what each agent is doing

## Features

- Dashboard showing all active agents
- Real-time status and activity logs
- Cost and token usage tracking
- Permission controls per agent
- Spawn new agents with custom configs
- Emergency kill switch
- Agent templates for common tasks

## Quick Start

```bash
npm install
npm start
# Open http://localhost:3000
```
