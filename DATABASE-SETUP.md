## Connect Web App to Database

### Manual Connection (Required Once)

1. Go to Railway dashboard: https://railway.app/project/43a7422e-842f-476e-a6bb-f5214e9a74a8
2. Click on **"agent-control-panel"** service (web app)
3. Click **"Variables"** tab
4. Click **"+ New Variable"**
5. Select **"Add a reference"**
6. Choose:
   - Service: **postgres**
   - Variable: **DATABASE_URL** (or DATABASE_PRIVATE_URL)
7. Name it: `DATABASE_PRIVATE_URL`
8. Click **Save**
9. Click **"Deploy"** or wait for auto-deploy

### Verify Connection

Once deployed, the app will automatically:
- Connect to PostgreSQL on startup
- Initialize database schema (tables)
- Use database for waitlist signups
- Fall back to in-memory if database unavailable
